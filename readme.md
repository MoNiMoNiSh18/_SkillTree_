# 🌳 SkillTree
AI-powered student placement and skill tracking platform.

## 🚀 Features

- AI Resume Analysis
- Placement Readiness Score
- Visual Skill Tree Roadmaps
- Daily Learning Tasks
- XP & Streak System
- Company Eligibility Checker
- JWT Authentication
- PDF Resume Parsing
- Gamified Progress Tracking
- Coding Practice Arena
## 🛠 Tech Stack

Frontend:
- React.js
- Tailwind CSS
- React Flow

Backend:
- Node.js
- Express.js

Database:
- MySQL

AI Features:
- PDF Parsing
- Skill Extraction
- Readiness Engine

## ⚙️ Installation

### Backend

cd server
npm install
npm start

### Frontend

cd client
npm install
npm start
